由開發者製作的桌寵外掛程式範例
* DemoClock: 為桌寵添加時鐘顯示的功能
* EdgeTTS: 桌寵說話的時候附帶語音,使用EdgeTTS

## VPet.Plugin.DemoClock
為桌寵增加時鐘顯示的功能,這算是程式碼嵌入類型MOD的DEMO

![democlock](democlock.png)

參考本軟體即可寫出自己的程式碼MOD

### 功能

為桌寵添加一個可以顯示時間的鐘錶

* 支援倒數計時
* 支援正計時
* 支持番茄鐘 (工作/休息)

每次使用番茄鐘完成工作,還可以獲得相應時間的 [金錢] 獎勵
